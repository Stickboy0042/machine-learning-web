# Machine Learning — Web Build

This repository publishes the browser build of **Machine Learning**, a Godot 4.7 puzzle game.

Play it at [stickboy0042.github.io/machine-learning-web](https://stickboy0042.github.io/machine-learning-web/).

The game is designed for desktop browsers with keyboard and mouse input. This public repository contains only the exported web build; the Godot project source is maintained separately.
